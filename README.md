# Dinero Web Front End
---

Front end Progressive web app for the dinero proof of concept MVP.  This web app is powered by react and redux.


## Commands
---

To install dependencies:
```shell
$ npm install
# or
$ yarn install
```

To run the dev server:
```shell
$ npm start
# or
$ yarn start
```

To test:
```shell
$ npm test
# or
$ yarn test
```

To build for production:
```shell
$ npm run build
# or
$ yarn run build
```
